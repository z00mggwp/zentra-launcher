# zentra-launcher
Zentra DarkRP oyuncuları artık launcher üzerinden;
› Sunucuya doğrudan bağlanabilir
› Güncelleme notlarını ve duyuruları takip edebilir
› İdari ekiple ilgili bilgilere ulaşabilir

Kurulum: İndirilen .exe dosyasını çalıştırmanız yeterlidir. Kurulum gerektirmez.

Windows SmartScreen uyarısı alırsanız "Daha fazla bilgi" → "Yine de çalıştır" seçeneğine tıklayınız.
